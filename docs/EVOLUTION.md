# 🚀 Plan d'Évolution - Scolaris

Ce document recense les améliorations stratégiques et techniques à implémenter pour faire passer Scolaris d'un prototype à une application de production robuste.

## ✅ Réalisé avec succès (Version 1.0)
- [x] **Stockage Cloud (Cloudinary)** : Migration hybride vers le cloud pour la persistance des fichiers.
- [x] **Haute Sécurité (Cookies HTTP-Only)** : Protection contre le vol de session (XSS).
- [x] **Refresh Tokens** : Système de maintien de session longue durée sécurisé.
- [x] **Mode Hors-ligne (PWA)** : Application installable et consultable sans internet.
- [x] **Export PDF Professionnel** : Bulletins et emplois du temps au format PDF.
- [x] **Dashboard Parent & Enseignant** : Statistiques, taux de réussite et suivi des signatures.
- [x] **Optimisation Assets** : Réduction du poids des images de 80%.
- [x] **Polissage Design** : Vitrine "Grand Public", Footer pro et réseaux sociaux (WA, IG).

## 🏗️ Architecture Backend & Performance (Prochainement)
- [ ] **Pagination Globalisée** : Support pour des milliers d'élèves.
- [ ] **Cache Redis** : Accélération de l'affichage des bulletins.
- [ ] **WebSockets** : Notifications instantanées en temps réel.

## 📊 Métier & Fonctionnalités Scolaires (Priorité V1.1)
- [ ] **Paiement Mobile Money (M-Pesa, Orange, Airtel)** : Pour les frais de scolarité.
- [ ] **Internationalisation (i18n)** : Support complet Lingala et Swahili.
- [ ] **Recours sur Note** : Messagerie dédiée pour les contestations.
