# Scolaris 🚀🌍

> **Statut de Production** : Déployé sur **Oracle Cloud (OCI)** à l'adresse `http://130.61.178.59`

**Scolaris** est la plateforme de gestion scolaire de nouvelle génération conçue pour l'éducation moderne. Alliant puissance administrative et intelligence artificielle, elle permet de gérer votre établissement avec une fluidité totale.

---

## 🏗️ Architecture Cloud (OCI)
Le système est propulsé par Docker avec 4 micro-services :
*   **Frontend** : Interface React ultra-rapide (Nginx).
*   **Backend** : Cerveau de l'application (Node.js/Express).
*   **Oracle IA** : Moteur hybride Gemini 1.5 Pro.
*   **Vault** : Sécurité et chiffrement des données sensibles.

## 🛠️ Guide Rapide
Toutes les commandes de gestion se trouvent dans le [Guide des Commandes](./docs/COMMANDS_GUIDE.md).

## 🧠 Intelligence Artificielle
Scolaris IA intègre un "Shield" de protection des données et une surveillance prédictive des serveurs 24/7.
